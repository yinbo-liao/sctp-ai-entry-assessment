# (SCTP) Advanced Professional Certificate in AI Engineering - Technical Entry Assessment

## Brief

Please click on the `<> CODE 🔻` button and select **"Download ZIP"** to download the files to your computer. Open the folder in your preferred IDE (e.g. VS Code, IntelliJ, PyCharm) and complete the questions inside the `src/` folder.

The entire assessment should not take you more than **60 – 90 minutes**.

---

## Expected Audience

Individuals looking to upskill or make a career switch into AI and software engineering roles. Potential roles include:

- AI Engineer
- AI Applications Engineer
- Full-Stack Developer (AI-driven)
- Backend / Frontend Developer
- Software Automation Engineer

---

## Language Choice

You may complete all questions in **any one** of the following programming languages:

- Python → use the `*_python.py` files
- JavaScript → use the `*_javascript.js` files
- Java → use the `*_java.java` files

Choose the language you are most comfortable with and use it consistently throughout.

---

## Problems

Please attempt to solve the problems in the following files inside the `src/` folder:

| Question | Topic | Files |
|----------|-------|-------|
| [Question 1](src/q1_python.py) | Functions & Conditionals | `q1_python.py` / `q1_javascript.js` / `Q1_java.java` |
| [Question 2](src/q2_python.py) | Arrays & Loops | `q2_python.py` / `q2_javascript.js` / `Q2_java.java` |
| [Question 3](src/q3_python.py) | String Manipulation | `q3_python.py` / `q3_javascript.js` / `Q3_java.java` |

Your code should be readable and include brief comments where helpful to explain your logic.

---

### 📝 Use of Generative AI Tools

Although AI assistance (e.g. ChatGPT, Copilot) is allowed, you are solely responsible for understanding all submitted code. Shortlisted candidates may be asked to walk through and explain their code at a face-to-face interview.

## Submission Instructions

⚠️ You must have a GitHub account to submit your assessment ⚠️

1. Create a free GitHub account at https://github.com if you do not already have one.
2. Create a new **public** repository named `sctp-ai-entry-assessment`.
3. Clone the repository to your local machine:
   ```
   git clone https://github.com/<your-username>/sctp-ai-entry-assessment.git
   ```
4. Copy your completed solution files into the cloned folder.
5. Commit and push your work:
   ```
   git add .
   git commit -m "Add solutions"
   git push
   ```
6. Submit the link to your GitHub repository via the application portal.

If you are unfamiliar with GitHub, these resources will help:

- [W3Schools Git Tutorial](https://www.w3schools.com/git/)
- [GitHub Docs: Getting Started](https://docs.github.com/en/get-started)

> The ability to self-learn is an important aspect of the SCTP course. Please go through the resources provided and familiarise yourself with the requirements to create the GitHub link for submission.
